Detailed Description

Hyper Gradient Descent (하이퍼 경사하강법) 최적화 알고리즘 구현

Hyper Gradient Descent는 기본 Gradient Descent의 고정된 보폭(Learning Rate) 또는 복잡한 Line Search 방법의 단점을 극복하기 위해, 보폭(alpha) 자체를 또 다른 경사하강법으로 학습(업데이트)하는 메타-최적화(Meta-optimization) 기법입니다.

현재 스텝의 기울기(Gradient)와 이전 스텝의 기울기의 내적(Dot product)을 이용하여 보폭을 적응적으로 조절합니다. (Algorithms for Optimization, Sec 5.9 참고)
