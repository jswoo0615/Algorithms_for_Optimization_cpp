Detailed Description

Primal-Dual Interior Point Method (내부점법) 기반의 Quadratic Programming (QP) Solver 구현

다음과 같은 형태의 2차 계획법(QP) 문제를 풉니다. Minimize 1/2 * x^T * P * x + q^T * x Subject to A_eq * x = b_eq A_ineq * x <= b_ineq

부등식 제약조건은 슬랙 변수(s)를 도입하여 A_ineq * x + s = b_ineq (s >= 0) 로 변환하여 처리합니다. 이 구현체는 KKT 조건을 풀기 위해 Newton-Raphson 기반의 Primal-Dual 스텝을 계산하며, 특이점(Singularity) 방지를 위한 Tikhonov 정규화와 발산 방지용 클램핑이 적용되어 있습니다.
