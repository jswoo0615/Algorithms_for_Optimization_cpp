Detailed Description

template<size_t N_vars, size_t N_ineq, size_t MaxNNZ_P, size_t MaxNNZ_A>
class Optimization::SparseIPMQPSolver< N_vars, N_ineq, MaxNNZ_P, MaxNNZ_A >

희소 행렬(Sparse Matrix) 기반 내부 점 기법 (Interior Point Method) QP 솔버

이 클래스는 다음과 같은 형태의 2차 계획법(Quadratic Programming) 문제를 풉니다. Minimize 1/2 * x^T * P * x + q^T * x Subject to A_ineq * x <= b_ineq

희소 행렬(Sparse Matrix) 연산과 Matrix-Free 켤레 기울기법(Conjugate Gradient)을 결합하여, 대규모의 제어 문제에서도 메모리를 적게 쓰고 빠르게 최적해를 계산합니다.

Template Parameters
N_vars최적화 변수의 개수 (Dimension of x)
N_ineq부등식 제약조건의 개수
MaxNNZ_P목적 함수 헤시안 P의 최대 비영요소(Non-zero) 허용 개수
MaxNNZ_A제약 조건 행렬 A_ineq의 최대 비영요소(Non-zero) 허용 개수

[Matrix-Free 연산] 축소된 KKT 시스템의 행렬-벡터 곱셈 (Schur Complement)

메모리에 거대한 KKT 행렬을 명시적으로 직접 만들지 않고 연산만 수행하는 기법입니다. 우리가 풀어야 할 축소된 헤시안 행렬은 H_sys = P + A^T * W * A 입니다.

Parameters
p곱할 입력 벡터 (탐색 방향)
W가중치 벡터 (대각 행렬 W = lambda * S^{-1} 의 대각 성분)
y계산 결과가 저장될 출력 벡터 (y = H_sys * p)
