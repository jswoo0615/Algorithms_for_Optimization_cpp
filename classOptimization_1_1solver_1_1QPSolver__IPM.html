Detailed Description

template<size_t Nx, size_t Nc>
class Optimization::solver::QPSolver_IPM< Nx, Nc >

고속 Primal-Dual Interior-Point Method (PDIPM) QP 솔버

min 0.5 * x^T * H * x + g^T * x s.t. C * x <= d (부등식 제약조건만 처리. 등식 제약조건은 Riccati 솔버가 동역학 단계에서 처리함)

슈어 보수(Schur Complement)를 활용하여 KKT 시스템을 압축하고, 초고속 LDLT 분해를 통해 탐색 방향을 도출합니다.
