Detailed Description

template<typename T, size_t Rows, size_t Cols>
class StaticMatrix< T, Rows, Cols >

전방 선언 (Forward Declaration)

Layer 1 & 2: Static Matrix Engine (Column-major)

Layer 1 - 기본 선형 대수 (산술 연산자, 메모리 접근자) Layer 2 - 분해 솔버 (LU / Cholesky / LDLT / MGS-QR / Householder-QR)

  • 블록 연산 (insert_block / insert_transposed_block / extract_block)

Cholesky 분해 (A = LL^T)

대각 원소: L[j][j] = sqrt(A[j][j] - Sigma L[j][k]^2) 비대각: L[i][j] = (A[i][j] - Sigma L[i][k]*L[j][k]) / L[j][j] (i > j)

Returns
true - 분해 성공 (A는 양정치)
false - d <= epsilon (비양정치 또는 특이행렬)

LDLT 분해 (Square-Root-Free Cholesky, A = L*D*L^T)

D 갱신: D[j][j] = A[j][j] - Sigma L[j][k]^2 * D[k][k] L 갱신: L[i][j] = (A[i][j] - Sigma L[i][k]*L[j][k]*D[k][k]) / D[j][j] (i > j)

수정 이력:

  • near_zero(D_jj) -> D_jj <= epsilon 으로 교체. near_zero는 절댓값 기준이어서 음수 D를 통과시키는 버그 존재. LDLT는 양정치 행렬 전용이므로 반드시 부호 검사가 필요하다.
Returns
true - 분해 성공
false - D <= epsilon (음정치 또는 특이행렬)

LU 분해 (In-place Doolittle)

A를 L·U로 in-place 분해한다. 대각 이상(j >= i): U 행 갱신 U[i][j] = A[i][j] - Sigma L[i][k]*U[k][j] 대각 이하(j > i): L 열 갱신 L[j][i] = (A[j][i] - Sigma L[j][k]*U[k][i]) / U[i][i]

Returns
true - 분해 성공
false - 대각 원소 ≈ 0 (특이행렬)

Householder QR 분해 (Compact 표현)

각 단계 i 에서 Householder 반사를 적용하여 i열 이하를 소거한다: norm_x = ||A[i:,i]||_2 v0 = A[i][i] + sign(A[i][i]) * norm_x v[k] = A[k][i] / v0 (k > i, compact 저장) tau = 2 / ||v||^2 A[i][i] = -sign * norm_x (R의 대각 원소) A <- (I - tau*v*v^T) A (rank-1 update, 열 단위)

수정 이력:

  • sqrt가 누산 루프 안에 중첩되어 있던 버그 수정: 누산 루프 완료 후 단 1회 sqrt를 적용하도록 분리.
Returns
true (영열 처리는 tau=0으로 스킵)

MGS-QR 분해 (Modified Gram-Schmidt)

열 단위 직교화 (in-place, *this 가 Q가 됨): R[i][i] = ||A[:,i]||_2 Q[:,i] = A[:,i] / R[i][i] R[i][j] = Q[:,i]^T * A[:,j] (j > i) A[:,j] -= R[i][j] * Q[:,i] (j > i)

수정 이력:

  • norm_sq = ... -> norm_sq += (누산 연산자 누락)
Returns
true - 분해 성공
false - 열 노름 ≈ 0 (선형종속)
