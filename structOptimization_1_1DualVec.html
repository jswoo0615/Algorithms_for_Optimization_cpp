Detailed Description

template<typename T, size_t N>
struct Optimization::DualVec< T, N >

N차원 다변수 함수 자동 미분을 위한 벡터 듀얼 구조체 (Gradient Vector 계산)

다변수 함수 f(x1, x2, ..., xn)에 대하여 함수 값과 N차원 기울기 벡터(Gradient)를 동시에 추적합니다. 스택 할당(std::array)을 사용하여 힙 메모리 할당으로 인한 오버헤드가 없습니다.
