Detailed Description

template<size_t Nx, size_t Nu>
struct Optimization::NMPCStorageData< Nx, Nu >

NMPC (Nonlinear Model Predictive Control) 최적화 문제에서 단일 예측 스텝 (t = k)의 데이터를 담는 블록 구조체.

LQR/Riccati 방정식 기반의 솔버에서 각 시점(stage)마다 필요한 선형화된 시스템 동역학, 2차 비용 함수의 가중치, 그리고 최적화 수행 후 얻어지는 제어 이득(gain) 행렬들을 한 곳에 모아 관리합니다. 이 구조체는 메모리를 연속적이고 정적으로 할당하기 위해 동적 할당 없이 배열(StaticMatrix/StaticVector)만을 사용합니다.

Template Parameters
Nx상태 변수(State variables)의 개수 (예: 위치, 속도, 각도 등)
Nu제어 변수(Control inputs)의 개수 (예: 조향각, 가속도 등)
